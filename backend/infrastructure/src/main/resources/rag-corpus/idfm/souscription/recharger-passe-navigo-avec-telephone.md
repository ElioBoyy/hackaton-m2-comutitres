# Comment recharger mon passe Navigo avec mon téléphone ?

- Titre : Comment recharger mon passe Navigo avec mon téléphone ?
- URL source : https://www.iledefrance-mobilites.fr/aide-et-contacts/chargement-passe-navigo-avec-un-telephone/comment-recharger-passe-navigo-avec-telephone
- Date de capture : 15/06/2026
- Mis à jour le 26 mai 2026

Découvrez comment recharger votre passe Navigo en vidéo sur Android et iOS.

- Dans le menu **Titres**, choisissez **>** Sur mon passe **Navigo** puis placez votre passe contre le dos du téléphone pour en consulter le contenu.
  À noter : sur les iPhones, vous devez cliquer sur « Lire mon passe » avant de présenter le passe en haut du téléphone.
- L'application affiche alors les titres et forfaits valides présents sur votre passe Navigo et vous permet d'acheter un titre si vous le souhaitez.
- L'application affiche ensuite les différents titres disponibles à l'achat. Vous sélectionnez le titre et, si nécessaire, la date, les zones et la quantité.
- Avant le paiement, vous êtes invité à vous authentifier sur Île-de-France Mobilités Connect afin de bénéficier de plusieurs avantages : enregistrement de votre moyen de paiement, justificatif d'achat nominatif, parcours d'achat simplifié, consultation de vos derniers achats, meilleure prise en charge de vos demandes SAV.
- Pour le paiement, vous devez disposer d'une carte bancaire et saisir une adresse e-mail à laquelle le justificatif d'achat vous sera envoyé.

**Dernière étape : revenez sur l'application !**

- Suite au paiement, vous devrez à nouveau présenter votre passe Navigo au dos du téléphone pour y charger le titre acheté. Vous êtes probablement passé dans votre application bancaire pour valider le paiement ; vous devez alors retourner sur l'application d'achat de titre afin de présenter votre passe pour y charger le titre acheté.

Vous pouvez alors utiliser votre passe Navigo pour voyager !

# Quels passes Navigo peuvent être rechargés avec un téléphone ?

- Titre : Quels passes Navigo peuvent être rechargés avec un téléphone ?
- URL source : https://www.iledefrance-mobilites.fr/aide-et-contacts/chargement-passe-navigo-avec-un-telephone/quels-passes-navigo-peuvent-etre-recharges-avec-un-telephone-nfc
- Date de capture : 15/06/2026
- Mis à jour le 10 mars 2025

Le service de rechargement avec votre téléphone NFC Android ou iOS est disponible pour tous les passes Navigo récents.

En fonction du type de passe dont vous disposez (passe Navigo, passe Navigo Easy, etc.), les titres proposés peuvent cependant différer. Vous pouvez consulter la page https://www.iledefrance-mobilites.fr/titres-et-tarifs/supports/smartphone, rubrique « Quel support pour quel titre ? » pour en savoir plus.

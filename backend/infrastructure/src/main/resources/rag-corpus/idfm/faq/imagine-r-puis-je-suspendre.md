# Puis-je suspendre mon forfait imagine R ?

- Titre : Puis-je suspendre mon forfait imagine R ?
- URL source : https://www.iledefrance-mobilites.fr/aide-et-contacts/forfait-imagine-r/puis-je-suspendre-mon-forfait-imagine-r
- Date de capture : 15/06/2026
- Mis à jour le 20 juin 2025

**La suspension du forfait imagine R n'est pas possible.**

**Il est possible de résilier le contrat imagine R sous certaines conditions** (voir : https://www.iledefrance-mobilites.fr/aide-et-contacts/forfait-imagine-r/comment-resilier-mon-forfait-imagine-r).

Notez qu'après une résiliation, le contrat ne peut pas être réactivé.
